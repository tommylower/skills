# skills

instruction files for AI coding tools — design systems, workflow patterns, and integration guides.

## structure

- `core/` — defaults: design system, motion, workflow rules
- `integrations/` — patterns for pencil, obsidian, framer
- `collected/` — skills from others

## usage

reference in your project's AI config file:

- claude code → `CLAUDE.md`
- cursor → `.cursorrules`
- windsurf → `.windsurfrules`
- copilot → `.github/copilot-instructions.md`
- codex → `.github/codex-instructions.md`
